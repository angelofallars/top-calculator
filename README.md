## Keybindings

- ( 0-9 ) - Numbers
- ( . ) - Degree symbol
- ( + / = ) - Addition
- ( - ) - Subtraction
- (x / \*) - Multiplication
- ( / ) - Division
- ( % ) - Modulo (Remainder)
- ( Enter ) - Calculate
- ( Backspace ) - Delete/Clear
